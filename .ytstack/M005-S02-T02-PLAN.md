## What this does

`JamQrOverlay`: fullscreen fixed overlay (light background for scan
contrast), renders the join URL as a large QR via `qrcode`'s `toDataURL`
(client-side, dynamic import keeps it out of the shared bundle), plus the
session name and the human-readable URL underneath. Esc / click / close
button dismisses. PartyHostView gets a "Show QR" button next to the join
URL card. Big-screen mode = the overlay itself (host fullscreens the tab).

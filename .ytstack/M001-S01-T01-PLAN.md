# M001-S01-T01 -- Task Plan

**Summary:** Routes-Audit. Alle `src/app/[locale]/*` Pages + Public-Surfaces (`/s/[slug]`, `/p/[slug]`, `/u/[handle]`, `/embed/*`) + `/api/*` Mounts enumerieren, pro Route Metadaten erfassen (Pfad, Page-File, Bounded Context, Auth, Top-Components). Output landet in `.ytstack/M001-ROUTE-CATALOG.md`.

## Files

**Create:**
- `.ytstack/M001-ROUTE-CATALOG.md` -- der Audit-Output. Drei Sektionen: (1) `[locale]/*` user-facing pages, (2) public surfaces ausserhalb locale, (3) `/api/*` mounts gruppiert nach Bounded Context.

**Read-only (Quellen):**
- `src/app/[locale]/**/page.tsx` (~25 Routes)
- `src/app/[locale]/layout.tsx` -- top-level AppShell wiring
- `src/app/s/[slug]/page.tsx` + `src/app/p/[slug]/page.tsx` + `src/app/u/[handle]/page.tsx` + `src/app/embed/**/page.tsx`
- `src/app/api/**/route.ts` -- ~80 route groups
- `src/middleware.ts` -- Auth-Gates pro Route-Pattern
- `.ytstack/FEATURE-MAP.md` §2 -- Bounded-Context-Labels reused
- `docs/feature-inventory.md` -- existing inventory zum Cross-Check

**Modify:** none.

**Delete:** none.

## What this does

Drei-Pass-Audit:

**Pass 1 -- User-facing pages:** `find src/app/[locale] -name "page.tsx"` → fuer jede gefundene Datei eine Zeile mit `URL-pfad | page.tsx Pfad | Bounded Context (aus FEATURE-MAP §2) | Auth required? (aus middleware.ts matcher + page-internal getServerSession) | Top-Components (top 3 imports aus dem JSX-Tree)`. Tabelle im Catalog.

**Pass 2 -- Public surfaces:** dasselbe Schema fuer `/s/[slug]`, `/p/[slug]`, `/u/[handle]`, `/embed/*`. Auth-Spalte = "public" wenn middleware sie passieren laesst. Anmerken: diese URLs sind Permalinks, nicht Teil interner Navigation, werden in M001-S02 (Journey "Share"-Pfad) referenziert.

**Pass 3 -- API mounts:** `find src/app/api -name "route.ts"` → grupiert nach Bounded Context aus FEATURE-MAP §2. Pro Route: HTTP-Methoden (GET/POST/PATCH/DELETE), Auth-Modus (session / API-Key / public / webhook), kurz was sie tut. Bounded Context als H2-Sektion, Routen als Tabelle darunter.

Keine Interpretation, keine Empfehlungen -- pure Inventur. Konsolidierungs-Vorschlaege kommen erst in S02.

```bash
  echo "PAGE_ROUTES_LISTED=$(grep -cE '^\| `/' .ytstack/M001-ROUTE-CATALOG.md)" && \
  echo "ACTUAL_PAGE_FILES=$(find src/app/\[locale\] src/app/s src/app/p src/app/u src/app/embed -name 'page.tsx' 2>/dev/null | wc -l | tr -d ' ')" && \

```

## What this does

**Pass A -- Viewport-conditional rendering inventory:** `grep -rn "useMediaQuery\|window.matchMedia\|matchMedia\|isMobile\|isDesktop" src` → Liste aller JS-Level viewport-branches. Plus `grep` fuer Tailwind `sm:`, `md:`, `lg:`, `xl:` Prefixes in JSX files (`*.tsx`) -- zaehlen, nicht enumerieren. Output: Tabelle "Component -> # JS-branches -> # tailwind-breakpoint uses".

**Pass B -- Mobile-specific components & flows:**
- `BottomSheet` Konsumenten und tatsaechliches Rendering-Conditional.
- `PullToRefreshContainer` Konsumenten.
- `ExpandedPlayer` -- wann triggert es, mobile-only?
- `KeyboardShortcutsModal` -- ist mobile-relevant?
- Mobile-Nav-Pattern in AppShell (sidebar collapse / bottom-nav?).

**Pass C -- PWA infrastructure:**
- `public/manifest.json` lesen, key fields enumerieren (name, icons, scope, start_url, display, shortcuts).
- `ServiceWorkerRegistrar.tsx` Registrations-Logik (siehe PWA-Memory: `/sw.js?v=<sha>` + BUILD_ID).
- `OfflineIndicator.tsx` -- wann zeigt es, was bedeutet "offline".
- Public manifest-Routen mountpath (`/manifest.json`, icons paths).
- Pre-deploy cache-busting infrastructure (per Memory: `NEXT_PUBLIC_BUILD_ID` → CI → Docker ARG → SW → namespace).

**Pass D -- Gesture / swipe / long-press patterns:**
- `SwipeablePlaylistItem.tsx`, `library/swipable-song-row.tsx` -- swipe handlers via was?
- Touch events: `grep -rn "onTouchStart\|onTouchEnd\|onTouchMove\|swipe\|onPointerDown" src/components`.
- Long-press: `grep -rn "onLongPress\|long-press\|longPress" src`.

**Pass E -- Offline / cache surfaces:**
- `useOfflineCache.ts` -- was wird offline-cached, wie viele Files referenzieren es.
- `lib/cache/offline.ts` -- API zu Storage (IndexedDB?).
- IndexedDB references: `grep -rn "indexedDB\|idb\|dexie" src`.
- `formatBytes` calls (Audit-Sentinel fuer offline-cache-Anzeige).

**Pass F -- Cross-cutting observations:** Erstes-Eindrucks-Sektion mit 5-10 Befunden fuer S02 (z.B. "ist Mobile-First, Desktop-Adapted, oder umgekehrt?", "ist BottomSheet under-utilised wenn nur 1 Consumer", "ist die Nav mobil noch 17-Items oder klappt sie zusammen").

## What this does

The page renders standalone (no AppShell — guests have no session): compact
header (session name, host name, budget countdown "N songs left"), a
now-playing card when the host plays a session song, and the request queue
(pending cards with prompt text + guest name + spinner; ready cards with
title/cover; failed marked honestly). Unknown token → friendly "this party
doesn't exist" state; closed session → "party ended" state. Dark-first,
violet accents (brand baseline). Composer (T02) and nickname (T03) follow.
